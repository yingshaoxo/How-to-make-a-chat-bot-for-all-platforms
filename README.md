## Tutorial overview

### 介绍
作为现代人或者程序员，如果我们不能对我们的生活有完全的掌控，那足以证明我们的失败。
在这里我们不仅仅是在构建一个聊天机器人，我们还是在构建一个总控机器人。
它不光可以接收你所有聊天平台的即时信息，还可以对其进行完全的掌控，如过滤、分组显示等。
这样，你便对你的生活多了一份掌控。

**学习前提**
* Python3
* Basic Linux commands
* And the most important thing is: have the desire to learn

**目标人群**
* 想要学习如何做自己的聊天机器人
* 想要从一个平台管理所有聊天平台的消息

**时间**

每周星期五下午5点( 预先录制的视频为主
